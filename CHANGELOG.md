## [1.0.0] - 2026-05-03

### Added
- Initial release of Suriname HR/Payroll module
- Core payroll processing functionality
- Suriname-compliant tax calculator (progressive brackets 0%-38%)
- Social security deduction calculations (employee 2.5%, employer 14-16%)
- Overtime calculation (1.5x standard, 2x for holidays)
- FVO (parental leave fund) declaration support
- Leave management (annual, sick, maternity)
- Employee profile with Suriname-specific fields
- Salary structure management
- Payroll entry processing
- Salary slip generation
- Monthly payroll reports
- Tax and compliance documentation
- Role-based permissions
- 7-year record retention support
- Comprehensive test suite
- Full API documentation
- Setup and installation guides
